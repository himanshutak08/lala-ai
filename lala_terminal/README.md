# Lala Terminal

Claude-style AI terminal for Home Assistant using Ollama.

## Features

- Sidebar terminal
- Ollama integration
- Local AI assistant
- Claude Code style experience
- Home Assistant aware AI

## Usage

Open:
- Home Assistant Sidebar
- Lala Terminal

Inside terminal:

```bash
lala
```

Example:

```txt
create automation to turn off lights at 11 PM
```
